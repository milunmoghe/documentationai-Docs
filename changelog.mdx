## Upgrading zapRF

Follow these steps to update your installation and migrate designs seamlessly.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your terminal.

    <CodeGroup tabs="Python,Node.js">
      ```python
      import zaprf
      print(zaprf.__version__)
      ```
      ```javascript
      const zaprf = require('zaprf-sdk');
      console.log(zaprf.version);
      ```
    </CodeGroup>
  </Step>
  <Step title="Update Package" icon="download">
    Use your package manager to install the latest version.

    <Tabs>
      <Tab title="pip" icon="python">
        ```bash
        pip install --upgrade zaprf-sdk
        ```
      </Tab>
      <Tab title="npm" icon="nodejs">
        ```bash
        npm update zaprf-sdk
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Validate Designs" icon="check-circle">
    Re-run simulations on active projects to confirm compatibility.

    <Callout kind="tip">
      Backup your design library before upgrading. Use `zaprf export --all` for safe archiving.
    </Callout>
  </Step>
</Steps>
